BMI Calculator
A simple, responsive web application built with HTML, CSS, and JavaScript that calculates a user's Body Mass Index (BMI) based on their height and weight input.

Overview
This project is a practical implementation of front-end web development skills. It takes user input for height and weight, applies the standard BMI formula, and provides immediate feedback on their weight category (Underweight, Normal, Overweight, Obese).

Features
Accurate BMI Calculation: Uses the standard formula to compute BMI.
User-Friendly Interface: Clean and intuitive design for easy input of height and weight.
Instant Results: Dynamic calculation and display of the BMI result and classification without a page refresh.
Responsive Design: The layout adjusts to fit various screen sizes, ensuring a seamless experience on desktops and mobile devices.
Health Classification: Provides a clear categorization of the result based on World Health Organization (WHO) guidelines.
